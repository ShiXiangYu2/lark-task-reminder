### Phase 3: CLI 命令

#### TASK-008: CLI 命令实现
**类型**: HITL (Human-in-the-Loop)
**优先级**: P0
**依赖**: TASK-001, TASK-003, TASK-005

**任务内容**:
实现以下子命令：

**list 命令**:
- 显示任务列表
- 支持 --status, --limit, --format 选项
- 支持彩色输出

**today 命令**:
- 显示今日任务
- 按截止时间排序
- 标注超期和紧急任务

**sync 命令**:
- 从飞书同步任务
- 显示同步状态
- 显示新增/更新/删除数量

**report 命令**:
- 生成每日/每周报告
- 支持 --type daily/weekly 选项
- 格式化输出

**notify 命令**:
- 发送提醒通知
- 支持 --urgent 选项
- 限制通知频率

**config 命令**:
- 查看/设置配置项
- 支持 list, get, set 子命令

**产出**:
- `src/cli/index.ts`
- `src/cli/commands/*.ts`
- `src/cli/formatter.ts`

**验收标准**:
- [ ] 所有子命令正常工作
- [ ] 帮助信息完整
- [ ] 错误处理友好
- [ ] 集成测试覆盖
